# Calculateur IMC Android (Kotlin)

Application Android développée en **Kotlin** permettant de calculer l'Indice de Masse Corporelle (IMC) à partir du poids et de la taille de l'utilisateur.

## Fonctionnalités

- Saisie du poids (kg)
- Saisie de la taille (m)
- Calcul automatique de l'IMC
- Affichage de la valeur de l'IMC
- Affichage de la catégorie correspondante :
  - Maigreur
  - Corpulence normale
  - Surpoids
  - Obésité
  - Obésité sévère

## Technologies utilisées

- Kotlin
- Android Studio
- XML (Interface utilisateur)
- ViewBinding

## Aperçu

L'application permet à l'utilisateur de :

1. Saisir son poids.
2. Saisir sa taille.
3. Cliquer sur **Calculer IMC**.
4. Visualiser son IMC ainsi que sa catégorie.
